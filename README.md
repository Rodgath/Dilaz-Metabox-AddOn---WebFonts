# Dilaz Metaboxes - WebFonts AddOn
Webfont fields addon for Dilaz Metaboxes.

## Supported Webfonts
* [Fontawesome Icons](https://github.com/ZeroWP/Plugin-Boilerplate/issues)
* [Foundation Icons]
* Material Design Icons
* Linea Icons
	* Arrows
	* Basic
	* Basic Elaboration
	* Ecommerce
	* Music
	* Software
	* Weather
